# 🚀 careerBoost AI Pro

CareerBoost AI Pro is a fully offline, rule-based AI career toolkit built with Python and Streamlit — no paid APIs, no internet required once installed.

**Tagline:** Land More Interviews. Build a Stronger Career.

## Features

- 📄 Resume Analyzer — scores impact, ATS-readiness, readability, and career readiness
- 🎯 ATS Job Match — compares your resume against any job description
- ✍️ Resume Rewrite — strengthens weak bullet points with better action verbs
- 🎤 Interview Coach — HR, technical, and behavioral questions with STAR method tips
- 🗺️ Career Roadmap — personalized 30-day plan to become interview-ready
- 📊 Career Dashboard — visual snapshot of resume quality and readiness
- 🛡️ Job Scam Detector — flags red flags in job postings
- 📥 Downloadable Reports — TXT and PDF export
- ⚡ Fully Offline (No paid APIs, no OpenAI/Claude/Gemini calls)

## Built With

- Python
- Streamlit
- pandas
- pdfplumber (PDF resume parsing)
- python-docx (Word resume parsing)
- fpdf2 (PDF report generation)
- Regular Expressions
- Rule-Based NLP

## Why careerBoost AI Pro?

CareerBoost AI Pro helps students and job seekers improve resumes, optimize ATS compatibility, identify suspicious job offers, and prepare for interviews using fast, privacy-friendly local analysis — no internet connection or paid AI subscription required.

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
```
